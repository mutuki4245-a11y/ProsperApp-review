## アプリの主な機能

- 営業準備: 営業日の開始、出勤キャストの登録、キャスト情報と商品情報の管理
- 伝票管理: 卓番ごとの伝票作成、客情報、指名、注文、退店処理
- 注文入力: 営業中伝票への商品追加、キャストバック対象の指定
- 会計: 小計、税サ込み合計、支払方法、現金受取額、釣銭の登録
- 締め作業(未実装): 勤怠入力、キャスト売上調整、経費領収書入力

## 技術構成
- ASP.NET Core Razor Pages
- Supabase PostgreSQL
- Supabase RPC中心のDBアクセス
- Google OAuth / Google Drive API
- Azure App Service
